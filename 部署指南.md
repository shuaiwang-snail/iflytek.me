# iflytek.me 部署指南

## 前置准备

1. GitHub 账号
2. Vercel 账号（可用 GitHub 登录）
3. Neon 账号（PostgreSQL 数据库）

---

## 第一步：准备代码

### 1.1 创建 GitHub 仓库

1. 登录 GitHub
2. 点击右上角 "+" → "New repository"
3. 仓库名：`iflytek-me`
4. 选择 "Public" 或 "Private"
5. 点击 "Create repository"

### 1.2 推送代码到 GitHub

```bash
cd /Users/wangshuai/.openclaw/workspace/iflytek-me

# 初始化 git（如果还没初始化）
git init

# 添加远程仓库（替换为你的仓库地址）
git remote add origin https://github.com/你的用户名/iflytek-me.git

# 添加所有文件
git add .

# 提交
git commit -m "Initial commit"

# 推送
git push -u origin main
```

---

## 第二步：创建数据库

### 2.1 注册 Neon

1. 访问 https://neon.tech
2. 用 GitHub 账号登录
3. 创建新项目

### 2.2 获取数据库连接字符串

1. 在 Neon 控制台，点击 "Connection Details"
2. 选择 "Prisma" 格式
3. 复制连接字符串，格式如下：
   ```
   postgresql://用户名:密码@主机名/数据库名?sslmode=require
   ```

---

## 第三步：部署到 Vercel

### 3.1 导入项目

1. 访问 https://vercel.com
2. 点击 "Add New Project"
3. 选择你的 `iflytek-me` 仓库
4. 点击 "Import"

### 3.2 配置环境变量

在 Vercel 项目设置中，添加以下环境变量：

| 变量名 | 值 | 说明 |
|-------|-----|------|
| `DATABASE_URL` | 从 Neon 复制的连接字符串 | 数据库连接 |
| `NEXTAUTH_SECRET` | 随机字符串（如：`your-secret-key-here`） | 加密密钥 |
| `NEXTAUTH_URL` | `https://iflytek.me` | 网站 URL |
| `BLOB_READ_WRITE_TOKEN` | 从 Vercel Blob 获取 | 图片存储 |
| `CRON_SECRET` | 随机字符串 | Cron 任务密钥 |
| `ADMIN_EMAIL` | `admin@iflytek.me` | 管理员邮箱 |
| `ADMIN_PASSWORD` | 你的密码 | 管理员密码 |

### 3.3 部署

点击 "Deploy" 按钮，等待部署完成。

---

## 第四步：配置数据库

### 4.1 推送数据库结构

在本地终端运行：

```bash
# 设置环境变量
export DATABASE_URL="你的Neon数据库连接字符串"

# 推送数据库结构
cd /Users/wangshuai/.openclaw/workspace/iflytek-me
npx prisma db push
```

### 4.2 创建管理员账号

```bash
# 设置环境变量
export DATABASE_URL="你的Neon数据库连接字符串"
export ADMIN_EMAIL="admin@iflytek.me"
export ADMIN_PASSWORD="你的密码"

# 运行初始化脚本
npx tsx scripts/init-db.ts
```

---

## 第五步：配置域名

### 5.1 在 Vercel 添加域名

1. 进入 Vercel 项目 → Settings → Domains
2. 输入 `iflytek.me`
3. 点击 "Add"

### 5.2 在域名服务商配置 DNS

根据 Vercel 提供的 DNS 记录，在你的域名服务商（如阿里云、腾讯云）添加：

**方式 A：使用 A 记录**
```
类型: A
主机记录: @
记录值: 76.76.21.21（Vercel 提供的 IP）
```

**方式 B：使用 CNAME 记录（推荐）**
```
类型: CNAME
主机记录: @
记录值: cname.vercel-dns.com
```

### 5.3 等待 DNS 生效

通常需要 5-30 分钟。

---

## 第六步：配置 Vercel Blob

1. 进入 Vercel 项目 → Storage
2. 点击 "Create Database"
3. 选择 "Blob"
4. 创建后，点击 "Connect"
5. 复制 `BLOB_READ_WRITE_TOKEN` 到环境变量

## 常见问题

### Q1: 数据库连接失败？
- 检查 `DATABASE_URL` 是否正确
- 确认 Neon 数据库允许外部连接

### Q2: 图片上传失败？
- 检查 `BLOB_READ_WRITE_TOKEN` 是否配置
- 确认 Vercel Blob 已创建并连接

### Q3: 定时任务不执行？
- 检查 `CRON_SECRET` 是否配置
- 在 Vercel 项目 → Settings → Cron Jobs 中查看任务状态
