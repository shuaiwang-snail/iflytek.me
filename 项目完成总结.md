## 📋 部署前准备

### 需要配置的环境变量

在 `.env.local` 和 Vercel 中配置：

```bash
# 数据库
DATABASE_URL="postgresql://..."

# NextAuth
NEXTAUTH_URL="https://iflytek.me"
NEXTAUTH_SECRET="随机密钥"

# Vercel Blob
BLOB_READ_WRITE_TOKEN="..."

# Cron 任务
CRON_SECRET="随机密钥"

# 管理员账号
ADMIN_EMAIL="admin@iflytek.me"
ADMIN_PASSWORD="***"
```

### 外部服务

1. **Neon** - PostgreSQL 数据库
2. **Vercel** - 托管和部署
3. **Vercel Blob** - 图片存储

- 在域名服务商配置 DNS 解析

```bash

npx prisma db push

# 创建管理员账号
npx tsx scripts/init-db.ts
```
