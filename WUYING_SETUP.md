# 无影云电脑详细设置指南

## 第一步：下载并安装客户端

### 1.1 下载客户端

**方式一：官网下载**
1. 访问：https://www.aliyun.com/product/ecs/wuying
2. 点击 **"立即下载"** 或 **"下载客户端"**
3. 选择对应系统版本：
   - Windows 版
   - macOS 版
   - iOS/Android 版（手机平板）

**方式二：直接下载链接**
```
Windows: https://client.wuying.aliyun.com/download/latest/windows
macOS: https://client.wuying.aliyun.com/download/latest/mac
```

### 1.2 安装客户端

**Windows：**
1. 下载 `AliyunWuyingClient.exe`
2. 双击运行安装程序
3. 按提示完成安装
4. 桌面会出现 **"无影云电脑"** 图标

**macOS：**
1. 下载 `AliyunWuyingClient.dmg`
2. 双击打开 DMG 文件
3. 将应用拖到 Applications 文件夹
4. 首次打开可能需要在 **系统设置 → 隐私与安全性** 中允许

---

## 第二步：登录客户端

### 2.1 打开客户端

1. 双击桌面 **"无影云电脑"** 图标
2. 等待客户端启动

### 2.2 登录账号

**方式一：阿里云账号登录**
1. 选择 **"阿里云账号"**
2. 输入阿里云账号和密码
3. 点击 **"登录"**

**方式二：扫码登录**
1. 选择 **"扫码登录"**
2. 使用阿里云 App 扫描二维码

### 2.3 选择组织/工作区

如果有多个组织，选择包含你的云电脑的那个

---

## 第三步：连接云电脑

### 3.1 找到云桌面

登录后，会显示你拥有的云桌面列表：
```
┌─────────────────────────────────────┐
│  我的云桌面                          │
│                                     │
│  ┌─────────┐                        │
│  │ 云桌面1 │  状态: 运行中           │
│  │         │  系统: Windows/Linux    │
│  │         │  [连接] [管理]          │
│  └─────────┘                        │
│                                     │
└─────────────────────────────────────┘
```

### 3.2 连接方式

**方式一：直接连接（推荐）**
1. 点击云桌面卡片上的 **"连接"** 按钮
2. 等待连接（约 10-30 秒）
3. 进入云桌面界面

**方式二：使用 VNC 连接**
1. 点击 **"管理"**
2. 选择 **"VNC 连接"**
3. 适合系统故障时紧急连接

## 第四步：更换系统镜像（Ubuntu）

### ⚠️ 常见问题：无法更换系统

**可能原因：**
1. 云桌面正在运行中
2. 没有停止云桌面
3. 权限不足
4. 镜像市场没有权限

### ✅ 正确操作步骤

#### 步骤 1：停止云桌面

**必须在客户端或控制台先停止云桌面！**

**方法一：客户端内操作**
1. 连接到云桌面
2. 点击顶部工具栏的 **"电源"** 图标
3. 选择 **"关机"** 或 **"重启"**
4. 等待完全关机

**方法二：阿里云控制台操作**
1. 访问 https://wuying.console.aliyun.com
2. 进入 **"桌面管理"**
3. 找到你的云桌面
4. 点击 **"更多"** → **"停止"**
5. 等待状态变为 **"已停止"**

#### 步骤 2：更换镜像

1. 在控制台中，找到已停止的云桌面
2. 点击 **"更多"** → **"更换镜像"**
   
   如果没有这个选项，尝试：
   - 点击云桌面名称进入详情页
   - 找到 **"镜像信息"** 区域
   - 点击 **"更换镜像"**

3. 在镜像市场选择：
   ```
   镜像类型: 公共镜像
   操作系统: Linux
   版本: Ubuntu 22.04 LTS 64位
   ```

4. 点击 **"确定"**

5. 等待更换完成（约 5-10 分钟）

#### 步骤 3：启动云桌面

1. 更换完成后，点击 **"启动"**
2. 等待状态变为 **"运行中"**
3. 使用客户端重新连接

## 第五步：如果还是无法更换系统

### 方案 A：联系管理员

如果是企业/组织账号，可能需要管理员权限：
1. 联系你们的阿里云管理员
2. 请求更换镜像权限
3. 或让管理员直接帮你更换

### 方案 B：使用现有系统部署

**如果是 Windows 云电脑，也可以部署！**

**Windows 部署步骤：**

1. **安装必要软件**
   - 下载 Git：https://git-scm.com/download/win
   - 下载 Node.js 20：https://nodejs.org/dist/v20.11.0/node-v20.11.0-x64.msi
   - 安装时全部默认下一步

2. **打开 PowerShell（管理员）**
   - 按 `Win + X`
   - 选择 **"Windows PowerShell (管理员)"**

3. **执行部署命令**
   ```powershell
   # 创建目录
   mkdir C:\apps
   cd C:\apps
   
   # 克隆代码
   git clone https://github.com/shuaiwang-snail/iflytek.me.git
   cd iflytek.me
   
   # 安装依赖
   npm install
   
   # 构建
   npm run build
   
   # 创建环境变量
   $envContent = @"
   DATABASE_URL=file:./prisma/prod.db
   NEXTAUTH_URL=http://localhost:3000
   NEXTAUTH_SECRET=your-secret-key
   ADMIN_EMAIL=admin@iflytek.me
   ADMIN_PASSWORD=***
   "@
   $envContent | Out-File -FilePath .env.production -Encoding utf8
   
   # 初始化数据库
   npx prisma migrate deploy
   npx tsx scripts/init-db.ts
   
   # 启动应用
   npm start
   ```

4. **配置内网穿透**
   - 下载花生壳 Windows 客户端
   - 配置内网映射：127.0.0.1:3000
   - 获得外网访问地址

### 方案 C：使用阿里云 ECS

如果无影云电脑限制太多，建议：
1. 购买阿里云 ECS（1核1G 即可，约 80元/月）
2. 选择 Ubuntu 22.04 镜像
3. 执行标准部署脚本

## 常见问题

### Q1: 客户端无法登录？
- 检查网络连接
- 确认阿里云账号密码正确
- 尝试扫码登录

### Q2: 找不到"更换镜像"按钮？
- 必须先停止云桌面
- 检查是否有管理员权限
- 联系组织管理员

### Q3: 更换镜像后无法启动？
- 等待 5-10 分钟
- 在控制台查看状态
- 尝试强制重启

### Q4: Ubuntu 系统密码是什么？
- 默认用户：root
- 密码：在控制台重置或查看
- 控制台 → 桌面详情 → 重置密码
